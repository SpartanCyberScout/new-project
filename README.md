# SpartanGeoINT Dashboard

Interactive geospatial intelligence platform for visualizing wireless security audits and regional surveillance infrastructure.

## 🛡 Features
* **Clustered Visualization:** FastMarkerCluster for high-density data.
* **Tactical UI:** Dark mode, Fullscreen, and Measure tools.
* **Data Sources:** WiGLE and Atlas of Surveillance integration.

## 🚀 Usage
1.  ALIAS => `start-map` (Activate .venv)
2. `python3 map.py` (Generate index.html)
3. `git add -f index.html && git commit -m "Update" && git push`
